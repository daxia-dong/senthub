# Contributing to SENTHUB

Thanks for your interest in contributing! 🦐

## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/my-feature`
3. Make your changes
4. Run `npm run build` to verify everything works
5. Commit and push
6. Open a Pull Request

## Code Style

- TypeScript strict mode
- Follow existing patterns (Tailwind CSS, functional components)
- Keep it simple

## Reporting Issues

Use the GitHub Issues tab. Bug reports and feature requests are welcome!
